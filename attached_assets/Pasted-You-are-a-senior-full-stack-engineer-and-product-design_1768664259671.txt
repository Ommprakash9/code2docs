================================================
CORE PRODUCT PURPOSE
================================================

Code2Docs allows developers to:
- Sign up and log in securely
- Paste a public GitHub repository URL
- Automatically generate professional documentation using AI
- View, copy, and download generated documents

Documentation types:
- README
- BRD (Business Requirements Document)
- TRD (Technical Requirements Document)
- TEST Strategy Document

- React
- TypeScript
- Tailwind CSS
- Framer Motion
- Dark theme by default

Backend:
- Node.js with Express
- GitHub REST API (to fetch repository files)
- OpenAI API (for documentation generation)

Auth:
- Email/password authentication (JWT-based)

Database:
- SQLite (simple, local, production-ready for demo)
- Store users, projects, and generated documents
